# С чего начать

Wikipefia — ваш академический помощник: база знаний со статьями, руководствами и ресурсами, организованными по предметам и преподавателям.

## Навигация

- **По предмету**: нажмите на название предмета, чтобы увидеть обзор, категории статей и информацию о преподавателях
- **По преподавателю**: нажмите на имя преподавателя, чтобы увидеть профиль, рейтинги, отзывы и статьи
- **Поиск**: нажмите **Cmd+K** / **Ctrl+K** для мгновенного поиска

## Настройки языка

Wikipefia поддерживает три языка: **русский** (по умолчанию), **английский** и **чешский**. Переключатель языка находится в верхней панели навигации.

<Callout type="warning">
Не все статьи доступны на всех языках. Если статья не переведена на ваш язык, вы увидите уведомление о том, на каком языке показан контент.
</Callout>

## Горячие клавиши

| Сочетание | Действие |
|---|---|
| `Cmd/Ctrl + K` | Открыть поиск |
| `Escape` | Закрыть поиск |
| `↑ / ↓` | Навигация по результатам |
| `Enter` | Перейти к результату |
